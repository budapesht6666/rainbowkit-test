# Рекомендации по использованию агентов для этого репозитория

Этот репозиторий содержит приложение Next.js, расположенное в корне этого репозитория.
При интерактивной работе над проектом с агентом следуйте приведенным ниже рекомендациям.

## 1. Правило локальной генерации кода

- Всегда следуйте стилю кода проекта, описанному в [codestyle.docs.md](codestyle.docs.md).
- Только TypeScript. Используйте импорт модулей ES и явные типы для публичных API.
- Используйте псевдоним пути `@/*`, как настроено в [tsconfig.json].
- Tailwind CSS: скомпонуйте классы через `cn` из [src/lib/utils.ts](src\lib\utils.ts); избегайте конфликтов классов.
- При добавлении или использовании новой библиотеки перед написанием кода загружайте актуальную документацию через Context7 mcp.
